# Personal Lawyer — Landing Page with Flask Backend

Веб-приложение для юридического бюро в Калининграде. Проект представляет собой адаптивный лендинг с интегрированной формой обратной связи и автоматической отправкой заявок на электронную почту.

## 🚀 Функционал
*   **Обработка форм:** Сбор данных пользователей (имя, email, телефон, сообщение) через POST-запросы.
*   **Email-интеграция:** Автоматическая отправка уведомлений о новых заявках через SMTP Gmail.
*   **SEO-оптимизация:** Настроены мета-теги Open Graph, Viewport для мобильных устройств, а также файл `favicon`.
*   **Безопасность:** Реализовано скрытие конфиденциальных данных (пароли, почта) с помощью переменных окружения (`.env`).
*   **Деплой:** Приложение полностью настроено для работы на хостинге Amvera с использованием Gunicorn.

## 🛠 Стек технологий
*   **Backend:** Python 3.10+, Flask
*   **Frontend:** HTML5, CSS3 (адаптивная верстка), Jinja2
*   **Server:** Gunicorn (WSGI HTTP Server)
*   **Инструменты:** Git, SMTP, python-dotenv
